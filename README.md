# AEOS Visualizer

A beautiful, interactive data visualization tool that transforms raw data into engaging animated infographics using AI-powered insights.

![AEOS Visualizer](https://ibb.co/MnwpdKM)

## Features

- 🎨 **Beautiful Visualizations**: Create pie charts, bar charts, and treemaps with stunning animations
- 🎯 **AI-Powered Insights**: Automatic data analysis and trend detection
- 🎬 **Video Export**: Export your visualizations as animated videos in multiple resolutions
- 📊 **Multiple Chart Types**: Choose from various chart types that best represent your data
- 🎨 **Customizable Themes**: Select from a variety of beautiful, pre-designed themes
- 📱 **Responsive Design**: Works seamlessly across all devices
- 🌙 **Dark Mode**: Built-in dark mode support for comfortable viewing

## Quick Start

```bash
# Clone the repository
https://github.com/harish-3/AEOS-Visualizer

# Navigate to project directory
cd aeos-visualizer

# Install dependencies
npm install

# Start development server
npm run dev
```

## Usage

1. **Input Data**
   - Enter percentages in text format (e.g., "30% for Category A, 40% for Category B")
   - Upload CSV or text files with percentage data
   - AI automatically processes and enhances your input

2. **Customize Visualization**
   - Choose from multiple chart types (Pie, Bar, Treemap)
   - Select from various color themes
   - Adjust animation settings

3. **Export Options**
   - Download as static image (PNG)
   - Export as animated video (WebM)
   - Save data as JSON

## Data Format

The visualizer accepts data in the following formats:

```text
// Text Input Examples
30% for Category A, 40% for Category B, 30% for Category C

// Alternative Format
Category A has 30%, Category B with 40%, Category C at 30%
```

## Video Export Options

- **Resolutions**: 4K, 2K, 1080p, 720p
- **Animation Types**: Scale, Rotate, Fade, Bounce, Wave, Pulse, Spiral, Swing
- **Duration**: 1-5 seconds
- **FPS**: 24-60 frames per second
